## Project Description

This is my full-stack application for CS208, built with Node.js, Express, Pug, and MariaDB (MySQL). I built a website for a donut shop called Downtown Donuts that allows users to view the menu, learn about the shop, and submit customer comments. The application uses Express for the backend, Pug for server-rendered views, and MariaDB for storing comments.

## Features

- Home page with a welcome message and call-to-action
- Menu page with donut and coffee items
- About page with shop background information
- Customer comments page with database storage
- Server-side validation for comment submissions
- Responsive design for mobile screens
- Friendly timestamp formatting for comments
- Pagination for comment listings
